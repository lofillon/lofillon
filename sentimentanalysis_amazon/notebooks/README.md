## Notebooks

- `knn_naive_bayes.ipynb`: dataset download + preprocessing + baseline + Naive Bayes
- `svm.ipynb`: SVM experiments (BoW / TF‑IDF / Word2Vec)
- `random_forest.ipynb`: Random Forest experiments (BoW / TF‑IDF / Word2Vec)
- `lstm.ipynb`: TensorFlow/Keras LSTM
- `bert.ipynb`: multilingual BERT sentiment model evaluation

All notebooks assume the raw dataset CSV is available at `../data/raw/french_to_english_product.csv`.
