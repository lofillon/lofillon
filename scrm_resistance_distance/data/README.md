## Dataset empirique (ports) — référence et installation

Le dataset utilisé dans l'étude empirique est fourni par Mendeley Data :

- **Global port supply-chains** (version 1, 2022)
- **DOI** : `10.17632/vzzy3b9gg4.1`
- **Lien** : `https://data.mendeley.com/datasets/vzzy3b9gg4/1`
- **Licence** : CC BY 4.0 (voir la page du dataset)

### Fichier requis

Télécharger le dataset et placer le fichier suivant dans le dépôt :

- `Global port supply-chains/Port_to_port_network/port_trade_network.csv`

Ce fichier n'est pas poussé sur GitHub car il dépasse 100 Mo.
